#### PocketShop (collaborative)

[View Frontend Repo](https://github.com/shaepy/pocketshop-frontend) ∙ [View Backend Repo](https://github.com/shaepy/pocketshop-backend) ∙ [Live Demo](https://pocketshop-app.netlify.app/)
